Lazurite Tx


INPUT

  • msg.payload

    • msg.payload is message body. It must be string

  • msg.dst_panid

    • msg.dst_panid is destination panid(16bit integer or the string). When it is undefined, setting in this node is used.

  • msg.dst_addr

    • msg.dst_addr is destination address(16bit integer or the string). When it is undefined, setting in this node is used.

  • msg.ackreg

    • msg.ackreq is boolean value. Enable/Disable ack request. When it is undefined, setting in this node is used.

  • msg.ret.success

    • msg.ret.success is true or false

    OUTPUT

  • msg.ret.rssi

    • msg.ret.rssi is rssi of ack

  • msg.ret.eack

    • msg.ret.eack is Uint8Array type data of enhance ack

  • msg.ret.errcode

    • msg.ret.errcode is error code, when send is failed.

  • msg.ret.errmsg

    • msg.ret.errmsg is error message, when send is failed.

    Lazurite Tx64


    INPUT

  • msg.payload

    • msg.payload is message body. It must be string

  • msg.dst_addr

    • msg.dst_addr is destination address(hex string of 64bit MAC or BigInt value)

  • msg.ackreg

    • msg.ackreq is boolean value. Enable/Disable ack request

  • msg.ret.success

    • msg.ret.success is true or false


    OUTPUT

  • msg.ret.rssi

    • msg.ret.rssi is rssi of ack

  • msg.ret.errcode

    • msg.ret.errcode is error code, when send is failed.

  • msg.ret.errmsg

    • msg.ret.errmsg is error message, when send is failed.

    Lazurite MAC

    get MAC Address of Lazurite.


    INPUT

    Just trigger of this process


    OUTPUT

    msg.payload.mac is 64bit MAC address

    msg.payload.panid is panid

    msg.payload.short is short address

    Lazurite RX


  • msg.payload

    • msg.payload is payload

  • msg.header

    • msg.header is MAC header of receiving packet

  • msg.seq_num

    • msg.seq_num is sequence number

  • msg.dst_panid

    • msg.dst_panid is panid

  • msg.dst_addr

    • msg.dst_addr is distination address

  • msg.src_addr

    • msg.src_addr is source address

  • msg.rssi

    • msg.rssi is rssi of receiving packet

  • msg.rxtime

    • msg.rxtime is receiving time
